## 一、实验目的

1. 掌握 Haar 小波分解方法。
2. 理解低频和高频子带含义。
3. 学会低频子带重构图像。
4. 比较不同层数压缩效果。
5. 分析压缩率与图像质量关系。

## 二、实验内容

读入 `cameraman.tif` 图像，调用 `dwt2()` 使用 Haar 小波进行二维小波变换。首先完成 1 级小波分解，得到 LL1、LH1、HL1、HH1 四个子带，并显示小波分解结果；然后仅保留 LL1 子带，将其余高频子带置零，利用 `idwt2()` 重构图像，并与原始图像进行对比分析。

在此基础上，对同一幅图像进一步进行 3 级 Haar 小波分解，得到 LL3 以及各层高频子带，显示多尺度小波分解结果；随后仅保留 LL3 子带，将其余各层高频子带置零，通过逐级逆小波变换重构图像。

最后对 1 级 LL1 重构结果与 3 级 LL3 重构结果进行比较，从保留系数比例、理想压缩比、MSE、PSNR、SSIM 和视觉效果等方面分析不同分解层数下图像压缩程度与重构图像质量的差异。

## 三、解题思路

首先读入 `cameraman.tif` 图像。若读入图像为彩色图像，则先转换为灰度图像；随后利用 `im2double()` 将图像转换为双精度类型，便于后续小波变换、误差计算和图像显示。实验采用 Haar 小波作为分解基，原因是 Haar 小波结构简单、运算直观，适合观察图像低频近似信息和高频细节信息的分离过程。

在 1 级小波分解部分，直接调用 `[LL1,LH1,HL1,HH1]=dwt2(I,'haar')` 对图像进行二维小波变换。LL1 表示图像的低频近似子带，主要保留图像整体亮度、主体轮廓和大尺度结构；LH1、HL1、HH1 为高频细节子带，分别反映不同方向上的边缘、纹理和灰度突变信息。为了便于观察分解结果，程序使用 `subplot` 将四个子带显示在同一窗口中。

在 1 级低频重构部分，仅保留 LL1 子带，将 LH1、HL1、HH1 全部置零，再调用 `idwt2()` 进行逆小波变换，得到只由低频近似信息重构出的图像。随后计算重构图像与原始图像之间的 MSE、PSNR 和 SSIM，同时统计保留系数数量、保留比例和理想压缩比，用于评价图像压缩程度和重构质量。

在 3 级小波分解部分，程序先对原图像进行第 1 层分解，得到 LL1 及第 1 层高频子带；再继续对 LL1 进行第 2 层分解，得到 A2 及第 2 层高频子带；最后继续对 A2 进行第 3 层分解，得到 LL3、LH3、HL3、HH3。这样可以得到从细尺度到粗尺度的多层小波表示。为了显示多尺度分解效果，程序将 LL3 放在系数图左上角，并按照小波分解常用布局放置各层高频子带。

在 3 级低频重构部分，仅保留最深层的 LL3 子带，将第 3 层、第 2 层和第 1 层的所有高频子带都置零，然后通过 `idwt2()` 逐级逆变换，先由 LL3 重构到第 2 层近似图像，再重构到第 1 层近似图像，最后恢复到原始图像大小。由于 LL3 是经过三次下采样后的最低频近似子带，保存的是更粗略的整体信息，因此压缩程度更高，但细节损失也更加明显。

最后将原始图像、1 级 LL1 重构图像、3 级 LL3 重构图像、两种误差图和 PSNR 柱状图放在同一窗口中显示。通过主观视觉效果和客观质量指标共同比较，可以分析分解层数增加后压缩比提高与图像质量下降之间的关系。

## 四、实验结果与讨论

程序运行后会输出三组图像结果和一组命令窗口文字结果。

第一组图像为 1 级 Haar 小波分解结果，包含 LL1、LH1、HL1、HH1 四个子图。LL1 子带仍能清楚表现原图像的主要结构和灰度分布，说明图像的大部分能量集中在低频部分；LH1、HL1、HH1 子带主要在人物边缘、衣服轮廓和背景灰度突变区域出现明显响应，说明高频子带主要保存边缘和纹理细节。

第二组图像为 3 级 Haar 多尺度小波分解结果。系数图左上角为 LL3 子带，表示经过多层分解后得到的低频近似信息；其周围和外层区域分别为第 3 层、第 2 层和第 1 层高频子带。通过该图可以看出，小波分解能够把图像信息按照尺度和方向进行分离，低频部分描述整体结构，高频部分描述不同尺度下的细节变化。

第三组图像为 1 级和 3 级低频重构结果比较，包括原始图像、仅保留 LL1 的重构图像、仅保留 LL3 的重构图像、两种误差图以及 PSNR 柱状图。仅保留 LL1 时，重构图像仍较清晰，主体轮廓和灰度层次保留较好，只是边缘锐度和细节纹理有所下降。仅保留 LL3 时，图像会明显变得平滑，边缘和纹理损失更明显，误差图中的差异区域也更突出。

命令窗口会输出两种重构方式的保留系数比例、理想压缩比、MSE、PSNR 和 SSIM。对于常见的 256×256 `cameraman.tif` 图像，1 级分解后的 LL1 大小通常为 128×128，仅保留 LL1 相当于保留原始系数数量的 25%，理想压缩比约为 4:1；3 级分解后的 LL3 大小通常为 32×32，仅保留 LL3 只保留原始系数数量的 1.56%，理想压缩比约为 64:1。

从结果可以看出，分解层数越高，仅保留最低频子带时保存的系数数量越少，压缩程度越高；但同时更多高频细节被舍弃，重构图像越平滑，边缘和纹理越模糊，MSE 通常增大，PSNR 和 SSIM 通常降低。因此，在基于小波变换的图像压缩中，需要根据实际需求在压缩率和重构图像质量之间进行折中选择。

## 五、实验总结

本次实验围绕图像压缩编码中的 Haar 小波变换展开，完成了 `cameraman.tif` 图像的 1 级和 3 级二维小波分解、子带显示、低频保留重构以及重构质量比较等内容。实验中通过 `dwt2()` 将图像分解为低频近似子带和多个方向的高频细节子带，再通过 `idwt2()` 在舍弃高频信息的条件下进行图像重构，直观展示了小波变换在图像压缩中的基本思想。

通过实验可以发现，LL 子带保存了图像的大部分能量和主要视觉结构，因此仅保留低频子带仍能重构出具有主体轮廓的图像；而高频子带主要保存边缘、纹理和细节，舍弃后会造成图像变平滑、边缘变模糊。1 级分解只保留 LL1 时压缩比相对较低，但重构质量较好；3 级分解只保留 LL3 时压缩比明显提高，但细节损失更严重，客观质量指标也会下降。本次实验加深了我对小波多尺度分析、图像低频与高频信息分布以及压缩率和重构质量之间关系的理解。

## 附录

### 1. 重要代码块展示

（1）读入图像并转换数据类型

```matlab
imageName = 'cameraman.tif';
I0 = imread(imageName);
if ndims(I0) == 3
    I0 = rgb2gray(I0);
end
I = im2double(I0);
waveName = 'haar';
```

（2）1级 Haar 小波分解

```matlab
[LL1, LH1, HL1, HH1] = dwt2(I, waveName);
```

（3）仅保留 LL1 子带进行重构

```matlab
zero1 = zeros(size(LL1));
I_rec1 = idwt2(LL1, zero1, zero1, zero1, waveName, size(I));
I_rec1 = limitRange(I_rec1);
```

（4）显示 1 级小波分解结果

```matlab
figure('Name', '1级 Haar 小波分解结果', 'Color', 'w');
subplot(2, 2, 1); imshow(LL1, []); title('LL1 低频近似');
subplot(2, 2, 2); imshow(LH1, []); title('LH1 水平细节');
subplot(2, 2, 3); imshow(HL1, []); title('HL1 垂直细节');
subplot(2, 2, 4); imshow(HH1, []); title('HH1 对角细节');
```

（5）3级 Haar 小波分解

```matlab
[A2, LH2, HL2, HH2] = dwt2(LL1, waveName);
[LL3, LH3, HL3, HH3] = dwt2(A2, waveName);
```

（6）仅保留 LL3 并逐级逆变换

```matlab
zero3 = zeros(size(LL3));
recA2 = idwt2(LL3, zero3, zero3, zero3, waveName, size(A2));
recA1 = idwt2(recA2, zeros(size(LH2)), zeros(size(HL2)), zeros(size(HH2)), waveName, size(LL1));
I_rec3 = idwt2(recA1, zeros(size(LH1)), zeros(size(HL1)), zeros(size(HH1)), waveName, size(I));
I_rec3 = limitRange(I_rec3);
```

（7）计算压缩比例与重构质量指标

```matlab
retainNum1 = numel(LL1);
retainRatio1 = retainNum1 / numel(I);
compressRatio1 = numel(I) / retainNum1;
[mse1, psnr1, ssim1] = calcQuality(I, I_rec1);

retainNum3 = numel(LL3);
retainRatio3 = retainNum3 / numel(I);
compressRatio3 = numel(I) / retainNum3;
[mse3, psnr3, ssim3] = calcQuality(I, I_rec3);
```

（8）显示 1 级与 3 级重构比较

```matlab
figure('Name', '1级与3级低频重构结果比较', 'Color', 'w');
subplot(2, 3, 1); imshow(I, []); title('原始图像');
subplot(2, 3, 2); imshow(I_rec1, []); title('仅保留 LL1 重构');
subplot(2, 3, 3); imshow(I_rec3, []); title('仅保留 LL3 重构');
subplot(2, 3, 4); imshow(abs(I - I_rec1), []); title('1级重构误差');
subplot(2, 3, 5); imshow(abs(I - I_rec3), []); title('3级重构误差');
subplot(2, 3, 6);
bar([psnr1, psnr3]);
set(gca, 'XTickLabel', {'1级LL1', '3级LL3'});
ylabel('PSNR / dB');
title('重构质量比较');
grid on;
```
